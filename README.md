# Singly Linked List Operations  

This repository contains basic implementations of **Singly Linked List (SLL)** in C++.  
Currently, it covers the following operations:  

- Insert a node  
- Delete a node  
- Display the list  

---

│-- insert.cpp
│-- delete.cpp

│-- main.cpp
